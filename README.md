# Harryson Nandwa GitHub Pages Site

This folder contains a ready-to-host static profile site for GitHub Pages.

## Files
- `index.html` — main page
- `style.css` — styling
- `profile-photo.png` — profile image
- `Harryson_Nandwa_CV.docx` — downloadable CV

## Publish on GitHub Pages
1. Sign in to GitHub.
2. Create a new public repository named **nandwa254.github.io**.
3. Upload all files from this folder into that repository.
4. Go to **Settings → Pages**.
5. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
6. Save.
7. Wait a minute or two.
8. Your site will be live at:
   - `https://nandwa254.github.io`

## Update later
- Edit `index.html` to change text.
- Edit `style.css` to change colors or layout.
- Replace `profile-photo.png` if you want a new profile image.
- Replace `Harryson_Nandwa_CV.docx` if you want a new CV file.
